# 民國七十年代第一國民健康操 AI 教練 (Cross-Device Web + Python Backend)

本專案是一個結合 1980 年代懷舊彩色電視機美學（CRT scanlines、木紋機身、LED 點陣計分板）與現代 AI 人體姿態估測技術（MediaPipe Pose）的跨裝置健康操互動系統。

### 🌟 系統特色
1.  **跨裝置算力分流**：使用網頁端 HTML5/CSS/JS 作為主要前端，將 AI 骨架運算與 8-bit 背景音樂合成完全交給使用者裝置（手機、平板、PC 瀏覽器）執行，實現極低延遲與流暢度。
2.  **共用 SQLite 資料庫**：後端採用 **FastAPI + SQLite**，支援會員帳號登入，在同一個區域網路下可共享所有裝置的運動紀錄。
3.  **自適應 AI 優化分析 (Adaptive AI)**：當您關閉程式或結束小節時，數據將**分節上傳**。後端會根據使用者的年齡層與運動表現，自動優化調整下蹲、伸展、轉體判定容差（ANGLE_TOLERANCE），實現個人化的教練引導。
4.  **重新編排 8 式動作**：新增「半深蹲」與「擴胸轉體」動作，節拍暫定為適合健康指引的 **60 BPM**。

## 🛠️ 安裝與啟動步驟

### 1. 建立並啟動虛擬環境 (Python)
專案目錄中已內建 `venv` 虛擬環境與 `requirements.txt`。請開啟 PowerShell 執行：
```powershell
# 進入專案目錄
cd C:\Users\Administrator\.gemini\antigravity\scratch\gymnastic-exercise-ai-coach

# 啟用虛擬環境
.\venv\Scripts\Activate.ps1

# 安裝後端必要的 FastAPI 與 Uvicorn 相依套件
pip install -r requirements.txt
```

### 2. 啟動後端伺服器
啟用虛擬環境後，直接執行 `main.py` 啟動後端服務：
```powershell
python main.py
```
伺服器將會啟動並監聽 `http://0.0.0.0:8000`。

### 3. 跨裝置存取 (PC & 手機)
*   **本機 PC 存取**：在 PC 的瀏覽器中輸入 `http://localhost:8000` 即可開啟 CRT 電視機前端。
*   **行動端 (手機/平板) 存取**：
    1. 確保您的手機與運行伺服器的 PC 連接在**同一個 Wi-Fi 區域網路**下。
    2. 查詢 PC 的區域網路 IP 地址（在 cmd 中執行 `ipconfig`，尋找 IPv4 位址，例如 `192.168.1.100`）。
    3. 在手機瀏覽器中輸入 `http://<您的PC_IP>:8000`。
    4. 手機端的前端介面會自動自適應為**手持復古遊戲機/掌機樣式**，讓您將手機靠在架子上，後退即可開始跳操！

## 🏃 8式核心動作與拍攝指南

為了讓 AI 能精準識別角度，當進行以下小節時，請配合畫面上的黃色/紅色警告框進行側身：
*   **正面 (Front View)** 動作：預備動作、第三節(上肢伸展)、第四節(擴胸轉體)、第五節(體側左右彎曲)、第七節(四肢開合)、第八節(整理運動)。
*   **側面 (Side View)** 動作：第一節(全深蹲)、第二節(半深蹲)、第六節(前後彎體)。

## 📊 自適應優化與資料庫運作說明
*   **登入/註冊**：進入網頁後會跳出 `INSERT COIN` 會員面板，註冊/登入後系統會加載該年齡群組的當前動作閾值 (例如: 60 歲以上長者會加載較寬鬆的深蹲容許度)。
*   **分節上傳**：每一節 (16拍) 結束時，前端會觸發換節哨音，並自動將該節的平均得分上傳至 SQLite 資料庫。
*   **離線處理與更新**：當上傳發生時，後端 `analyzer.py` 會被動觸發，針對使用者的群組數據進行加總分析。若發現某個群組的平均得分低於 75 分，將會在資料庫中動態將該群組的容差放寬，並在使用者下一次登入/重新載入時生效。
